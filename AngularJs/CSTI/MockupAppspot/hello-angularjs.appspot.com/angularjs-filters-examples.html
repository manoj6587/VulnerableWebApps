AngularJS Filter Examples & Code Samples

This example demonstrates various different filter options in Angular such as following:
  • Uppercase
  • Lowercase
  • Currency
  • Date
  • Filter
  • OrderBy
  • Linky

Uppercase Demo

{{firstname|uppercase}}
Type & Watch the text being written with UPPERCASE.
Code: {{firstname|uppercase}}. firstname is the model name and uppercase is the filter

Lowercase Demo

{{lastname|lowercase}}
Type & Watch the text being written with LOWERCASE.
Code: {{lastname|lowercase}}. lastname is the model name and lowercase is the filter

Currency Demo

{{salary| currency }}
Type & Watch the text being written with default currency (USD).
Code: {{salary| currency }}. salary is the model name and currency is the filter

Date Demo

{{dob| date:'medium' }}
Type & Watch the date being written in en_US locale.
Code: {{ dob| date:'medium' }}. dob is the model name and date is the filter with 'medium' representing the en_US locale

Filter Demo

{{cityObj.name}}
Code: cityObj in cities | filter: city. cities is the model name and filter is the filter

OrderBy Demo

City Name
{{cityObj.name}} 
<table>
	<tr>
		<td><a href="" ng-click="orderByField='name';reverse = !reverse">City Name</a></td>
	</tr>
	<tr ng-repeat="cityObj in cities | orderBy:orderByField:reverse">
