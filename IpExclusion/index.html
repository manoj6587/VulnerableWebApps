@@@@Target setup for IP exclusion feature testing@@@@

  1. Some configuration are need to check In case of target is down, steps are given below
  2. Make sure that eth0,ifcfg-eth2:2,ifcfg-eth2:1 and ifcfg-eth2:3 are up and running
  3. If above interfaces are down then make it up by running ifup ifcfg-eth2:2,ifup ifcfg-eth2:1,ifup ifcfg-eth2:3 command
  4. Check all interfaces are up and running using ifconfig command
